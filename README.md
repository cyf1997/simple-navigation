# simple-navigation
a simple navigation website for your team,implemented entirely on the frontend,can only use URL format.

**Technical Component**

- Vue3
- Vite7
- navie-ui 
- tailwindcss4
- ion,icon5



## USE
### prepare
- node v22+
- npm  v10+

### install

```cmd
npm install
```

### RUN

```
npm run dev
```

### PACKAGE

```
npm run build
```

## DEPLOY

First,you need prepate web server.For example Nginx.Then use dist directory  from following command：

```
npm run build
```





## OTHER

### config file
The file is in /public/data/data.json,it is data sourceyou for this project.You can change yourself config in it.

### ICON

All the ICONS of ionicons5 have been realized at present.

In other words, all icons from ionicons5 are already loaded when the project is initialized, so you can use an icon simply by using its name.

link：[xicons - Icons for Vue & React](https://www.xicons.org/#/)
